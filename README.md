# Pizza Parlor
 
  ### By Beverly Potts
 
## Description
 
  This web page simulates the pizza ordering process online by asking the user to select a size and one or more toppings, then displaying the selections and calculating a price total!
 
## Languages & Technologies Used:
 
  ### Programming Languages & Libraries
  * HTML
  * CSS
  * Bootstrap
  * JQuery
  * Javascript
 
## Installation
 
  1. Open a web browser, such as Google Chrome, and navigate to https://github.com/BAPotts/pizza-parlor .
  3. Click the green "Clone or download" button at the right of the screen.
  4. Select "Download ZIP."
  5. Use a file extractor or unzip program (such as PeaZip, Unzipper, WinZip, Zipware, or 7-ZIP) to extract the ZIP files onto your computer.
  6. Open the directory containing the extracted files.
  7. Double-click the "index.html" file to run the program through your web browser.
